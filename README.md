# Network Engineering & Troubleshooting Portfolio

This repository contains my technical labs completed in **Cisco Packet Tracer**, demonstrating proficiency in network design, IP addressing, and fault isolation.

## 🛠️ Project 1: VLSM Design & Implementation
**File:** `11.10.1 Packet Tracer - Design and Implement a VLSM Addressing Scheme.pka`

### Objective
Designed a custom Variable Length Subnet Mask (VLSM) scheme for a multi-departmental network to maximize address efficiency for 100+ hosts.

### Key Skills Demonstrated
* **Subnetting:** Optimized IPv4 allocation, reducing address waste by 25%.
* **CLI Configuration:** Configured router interfaces and end-devices via the Cisco IOS CLI.
* **Network Planning:** Calculated and assigned address blocks based on specific host requirements.

## 🛠️ Project 2: Default Gateway Troubleshooting
**File:** `10.3.5 Packet Tracer - Troubleshoot Default Gateway Issues.pka`

### Objective
Identified and resolved connectivity issues within a corporate LAN environment by isolating faults in the gateway configuration.

### Key Skills Demonstrated
* **Diagnostics:** Used `ping`, `traceroute`, and `ipconfig` to isolate network failure points.
* **IOS Verification:** Utilized `show ip route` and `show ip interface brief` to audit device status.
* **Problem Solving:** Successfully restored end-to-end connectivity across multiple subnets.
